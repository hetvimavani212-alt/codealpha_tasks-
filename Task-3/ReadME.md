# Task 3: Move JPG Files Using Python

## 📌 Project Description

This Python script automates the task of moving all **`.jpg`** image files from one folder (**source_images**) to another folder (**moved_images**).

The program checks each file in the source folder and moves only files with the `.jpg` extension. If the destination folder does not exist, it is created automatically.

---

## 🎯 Objective

Automate a repetitive file management task using Python.

---

## 🛠 Technologies Used

* Python 3
* `os` module
* `shutil` module

---


---

## 🚀 How to Run

1. Install Python 3 if it is not already installed.
2. Create a folder named **source_images**.
3. Place one or more `.jpg` files inside the **source_images** folder.
4. Save the Python script with .py extension.
5. Open a terminal or command prompt in the project folder.
6. Run the following command:

```bash
python move_jpg.py
```
