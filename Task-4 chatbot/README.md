# 🤖 Basic Rule-Based Chatbot (Python)

A simple Python chatbot that responds to predefined user inputs using `if-elif` conditions. This beginner-friendly project demonstrates the basics of Python programming, including functions, loops, conditional statements, and user input/output.

## 📌 Project Objective

Build a simple rule-based chatbot that can respond to common user messages such as:

- `hello`
- `how are you`
- `bye`

The chatbot provides predefined responses and continues the conversation until the user types `bye`.

## ✨ Features

- Greets the user.
- Responds to basic questions.
- Handles unknown inputs with a default reply.
- Runs continuously until the user exits.
- Beginner-friendly Python code.

## 🛠️ Technologies Used

- Python 3
- Functions
- `if-elif-else` statements
- `while` loop
- User Input (`input()`)
- Output (`print()`)

## 🚀 How to Run

1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.

```bash
git clone https://github.com/your-username/Basic-Chatbot.git
```

3. Navigate to the project folder.

```bash
cd Basic-Chatbot
```

4. Run the Python program.

```bash
python chatbot.py
```
